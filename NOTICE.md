# Third-party data

This project evaluates against the **Contract Understanding Atticus Dataset (CUAD) v1**,
created by The Atticus Project, licensed **CC BY 4.0**
(https://creativecommons.org/licenses/by/4.0/). Source: https://www.atticusprojectai.org/cuad.

`data/cuad/subset.json` is a derived subset (6 clause categories, span labels retained)
of CUAD v1, redistributed under the same CC BY 4.0 license with attribution.
No modification is made to the underlying contract text.
